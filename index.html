I am using colors to show how much visual power they have.

The circles above are your basic HTML div containers that have been transformed by using a blend of HTML and CSS properties. The roundness is achieved using the CSS property radius: 50%;. Additional visual impact is acheived with the use of 'SKEUMORPHISM'. Skeumorphism is often used to give a webpage click on object the appearance of a button. Or give an element a 3-dimensional effect, and provides a great UX by pulling the viewer in. Additional CSS styles add subtle shading, shadows and roundness, to give the circles a sophiticated 'marble' appearance. The combination makes the color marbles really pop off the page.

These circles above are used to provide examples of 'NAMED COLORS', and equivalents in 'HEXIDECIMAL", and 'RGB/RGBA', with the smaller 'marbles' showing how those colors render. The three large 'marble' circles are 'Tomato'. And the 5 smaller marbles comprise the other 5 named colors, CornFlowerBlue, LightSeaGreen, MediumSlateBlue, CornSilk and DarkOrange.

To the right displays a webpage navigational feature called a 'breadcrumb' using a chevron '>' pointer, (structured with HTML and styled with CSS), the breadcrumb 'location' is 'FONTS' crumbing to 6 types of font choices.

The breadcrumb font 'attributes' are Roboto, Betania Patmos In, Bitcount, Montserrat, Barlow, and Helvetica.

The first 5 fonts are Google fonts API, which require external loading from a web-based library, and ensures consistent rendering across devices. The API fonts offer more size and styling choices. Helvetica is a font pre-installed with Windows locally. These local fonts are system dependent, and limited for styling. Other pre-installed fonts include Times New Roman and, Verdana, more recognized and widely used.
